# 📊 Customer Churn Analysis using SQL & Power BI

## 📌 Project Overview

This project analyzes customer churn behavior using SQL and Power BI. The goal is to identify key factors affecting customer attrition and provide insights to improve customer retention.

---

## 🎯 Objectives

* Calculate and analyze customer churn rate
* Identify factors influencing churn
* Perform data cleaning and transformation using SQL
* Build an interactive dashboard using Power BI
* Generate actionable business insights

---

## 🛠️ Tools & Technologies

* SQL (MySQL)
* Power BI


---

## 📂 Dataset

* Telco Customer Churn Dataset
* ~7000+ customer records
* Features include:

  * Customer demographics
  * Services subscribed
  * Billing information
  * Churn status

---

## 🧹 Data Cleaning (SQL)

* Handled missing values in `TotalCharges`
* Converted data types to numeric format
* Checked for duplicate records
* Ensured data consistency

---

## 📊 Analysis Performed

* Total customers count
* Churn rate calculation
* Churn distribution (Yes/No)
* Churn by contract type
* Monthly charges vs churn
* Customer tenure analysis

---

## 📈 Dashboard Features

* KPI Cards:

  * Total Customers
  * Churn Customers
  * Churn Rate
  * Average Monthly Charges

* Visualizations:

  * Churn Distribution (Pie Chart)
  * Churn by Contract Type (Bar Chart)
  * Monthly Charges vs Churn (Column Chart)
  * Tenure vs Churn (Line Chart)

* Filters:

  * Gender
  * Contract
  * Internet Service

---

## 🔍 Key Insights

* Month-to-month contract customers have the highest churn
* Higher monthly charges are linked with higher churn
* Long-tenure customers are less likely to churn
* Certain payment methods show higher churn behavior
